# Welcome to your Stack Auth + Convex + Next.js app

This is a template for a new Stack Auth + Convex + Next.js app.

## Getting started

1. Clone the repository
    ```bash
    git clone https://github.com/stack-auth/convex-next-template.git
    ```

2. Install dependencies
    ```bash
    cd convex-next-template
    npm install
    ```

3. Register an account on [Stack Auth](https://app.stack-auth.com), copy the keys from the dashboard, and paste them into the `.env` file

4. Start the development server and go to [http://localhost:3000](http://localhost:3000)
    ```bash
    npm run dev 
    ```

5. Go to [https://dashboard.convex.dev](https://dashboard.convex.dev/) and select the project you just connected. Click "Settings", then "Environment Variables" and add your stack auth environment variables from step 3.
